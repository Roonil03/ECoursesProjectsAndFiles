# N-by-N Tic-Tac-Toe

> This advanced N-by-N Tic-Tac-Toe application is inspired by the principles taught in the Coursera course 'Learn React Hooks: Build a Tic-Tac-Toe Game' (https://www.coursera.org/learn/learn-react-hooks-build-a-tic-tac-toe-game). It scales the core state management concepts of React Hooks to support dynamic matrix sizes and heuristic-driven AI bots, offering an optimized, browser-native gameplay experience.

## Features
- **Dynamic Board Sizes:** Play on a 3x3, 4x4, or 5x5 grid.
- **Game Modes:** Play against a friend locally or challenge the AI bot.
- **AI Bot Options:** Easy mode (random plays) or Hard mode (Minimax algorithm with Alpha-Beta pruning & Heuristics for instantaneous responses).
- **Modern UI:** Built with Tailwind CSS, utilizing vibrant colors and smooth animations via `framer-motion`.
